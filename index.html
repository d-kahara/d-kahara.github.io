Projects

HairQ

An application commissioned by a client to provide services for salon owners in the UK to track their customers and provide a platform to advertise their businesses. Complementary android app built for users to access the platform on their mobile devices. The platform also allows users to book services in advance and also rate the quality of the services offered.

Backend built using Nodejs and Express.js for its minimalistic yet robust features

Database used is MongoDB for its useful built-in geolocation functions

Search and auto-complete functions implemented using Algolia

Customized calendar to allow bookings on the platform

Custom android webview to allow users access the platform on android phones.

NB: Currently building a mobile version of the app using django and react native.
